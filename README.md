# Portfolio — Mathieu HABIB
> Développeur Fullstack Freelance

## 🚀 Stack
- HTML / CSS / JS vanilla
- Nginx (serving)
- Docker

## 🌐 Live
[portfolio-mathieuhabib.vercel.app](https://portfolio-mathieuhabib.vercel.app)

## 🐳 Lancer en local avec Docker

```bash
# Build et lancement
docker-compose up --build

# Accès
http://localhost:8080
```

## 💻 Lancer sans Docker

Ouvre simplement `index.html` dans ton navigateur.

## 📦 Déploiement

Le projet est déployé automatiquement sur **Vercel** à chaque `git push` sur la branche `main`.
